# RAG Web Application

A RAG (Retrieval-Augmented Generation) website built with:

- **Backend**: FastAPI, LangChain v1.0, ChromaDB, SQLite
- **Frontend**: Vue 3, Pinia, Tailwind CSS

## Quick Start

### Backend

```bash
pip install -r backend/requirements.txt
uvicorn backend.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

See `walkthrough.md` for more details.
